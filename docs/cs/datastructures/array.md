# Array

A collection of elements stored at contiguous memory locations.

- Fixed size

## Operations

Create
```java
    int[] arr = new int[5];
```

```java
    int[] arr = {1,2,3,4,5};
```

```java
    int[] arr = new int[1];
```

Assign
```java
    arr[0] = 99;
```

Access
```java
    arr[0];
```

Modify Element
```java
    arr[0] = 2;
```

Multi-Dimensional Array
```java
    int[][] arr = new int[5][5];
```
